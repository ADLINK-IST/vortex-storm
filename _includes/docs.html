Get Started

Create configuration

In order to be able to create either a Spout or Bolt you need a configuration object that describes the type of data be read or written from Vortex.

VortexConfig<ShapeType> vortexConfig = 
    new VortexConfig(0, "Circle", ShapeType.class, "ShapeType",
                        new ShapeBuilder(), 
                        new String[]{"color", "x", "y", "shapeSize"}, 100);

Create Spout

To consume data from Vortex we need to create a Spout.

topologyBuilder.setSpout(
                "vortex-spout", new VortexSpout(vortexConfig), 1);

Create Bolt

To write data to Vortex we need to create a Bolt.

topologyBuilder.setBolt("vortex-writer", new VortexWriterBolt(writerConfig), 1);
