What is vortex-storm?

vortex-storm is a Java library that makes it possible to perform distributed data analytics and processing on PrismTech's Vortex IoT and IIoT data with Apache Storm. The library provides a Spout for ingesting Vortex data into Storm and a Bolt for writing data back into Vortex.

Distributed

Leverage the power of Apache Storm's distributed computing platform to add scalability to your analytics.

Real-time

Process and analyze the data to make decisions in real-time instead of waiting for a batch process to make the decision.

Easy to use

The library has been carefully written to make working with Vortex data in Storm as easy as possible.
